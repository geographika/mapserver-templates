Mapserver entry page

Mapserver provides OGC web-services that can be used to pull soil data directly from ISRIC servers (Please check ISRIC's data policy, disclaimer and citation).


GIS softwares like QGIS and ArcGIS can be used to download data for specific areas, together with links bellow indicated.


For data citation and license, please consult our: Data and Software Policy

Please check the ISRIC tutorial on how to access layers:

WMS: https://www.isric.org/instruction-wms

WCS: https://www.isric.org/web-coverage-services-wcs
